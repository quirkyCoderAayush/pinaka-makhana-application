## 📦 **INSTALLATION & SETUP**

### Frontend Setup
```bash
cd pinaka-makhana-frontend/makhana-store-frontend
npm install
npm run dev
```

### Backend Setup
```bash
cd pinaka-makhana-backend/makhana-store
mvn clean install
mvn spring-boot:run
```
